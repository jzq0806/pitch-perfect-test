# 🎵 Pitch Perfect Test - 音感测试网站

一个现代化的音乐听音测试网站，帮助用户测试和评估自己的音感能力。

## ✨ 功能特点

### 8个测试关卡

1. **Level 1: 单音识别 - C大调**
   - 播放C大调音阶后，识别随机播放的单个音符
   - 适合初学者

2. **Level 2: 音高对比**
   - 判断两个音符的音高关系（更高/相同/更低）
   - 训练相对音感

3. **Level 3: 单音识别 - 其他调**
   - 在G大调、F大调等不同调式中识别音符
   - 提升调式适应能力

4. **Level 4: 音高距离感知**
   - 判断两个音符之间的距离（很近/中等/很远）
   - 无需音乐理论知识

5. **Level 5: 旋律记忆**
   - 听两段旋律，判断是否相同
   - 测试听觉记忆能力

6. **Level 6: 双音同时识别**
   - 识别同时播放的两个音符
   - 挑战和声听辨能力

7. **Level 7: 节奏识别**
   - 识别不同的节奏型
   - 测试节奏感知能力

8. **Level 8: 绝对音高挑战**
   - 无参考音阶，直接识别音符
   - 终极挑战，测试绝对音感

### 核心特性

- ✅ **2次复活机会**：每次测试有2次答错机会
- ✅ **自动播放**：音频自动播放，无法重复（真实测试环境）
- ✅ **实时统计**：记录正确率、反应时间等数据
- ✅ **历史记录**：保存所有测试成绩，追踪进步
- ✅ **响应式设计**：支持手机、平板、电脑
- ✅ **纯前端实现**：无需服务器，使用 Web Audio API 生成音频

## 🎮 使用说明

### 开始测试

1. 在主页选择一个关卡
2. 点击关卡按钮进入测试
3. 音频会自动播放（先播放参考音阶，再播放测试音）
4. 从选项中选择你认为正确的答案
5. 点击 "Next" 按钮提交答案

### 答题规则

- 每个关卡有 **10道题**
- 你有 **2次复活机会**（答错2次后测试结束）
- 音频只播放一次，无法重复
- 选择答案后可以再次点击取消选择

### 查看结果

测试结束后会自动跳转到结果页面，显示：
- 正确率
- 答对题数
- 平均反应时间
- 使用的生命数
- 达到的关卡

### Dashboard

点击导航栏的 "DASHBOARD" 查看：
- 总测试次数
- 平均正确率
- 平均反应时间
- 最佳成绩
- 完整的测试历史记录

## 🎨 设计理念

- **简洁直观**：参考 Human Benchmark 的设计风格
- **泡泡选项**：音符选项使用圆形泡泡设计，美观易用
- **渐变背景**：紫色渐变背景营造专业氛围
- **流畅动画**：平滑的过渡和悬停效果
- **无干扰**：测试过程中无多余元素，专注答题

## 🔧 技术栈

- **HTML5**：页面结构
- **CSS3**：样式和动画
- **JavaScript (ES6+)**：游戏逻辑
- **Web Audio API**：实时音频生成
- **LocalStorage**：本地数据存储

## 📊 数据存储

所有数据存储在浏览器的 LocalStorage 中：

- `pitchTestResults`：当前测试结果
- `pitchTestHistory`：历史测试记录

**注意**：清除浏览器数据会删除所有记录。
